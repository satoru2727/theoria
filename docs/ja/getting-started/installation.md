# インストール

## 要件

- Python 3.11以上
- pip または uv

## ソースからインストール

```bash
git clone https://github.com/theoria-project/theoria.git
cd theoria
pip install -e ".[dev]"
```

## インストール確認

```bash
theoria version
```
