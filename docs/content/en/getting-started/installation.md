# Installation

## Requirements

- Python 3.11 or higher
- pip or uv

## Install from Source

```bash
git clone https://github.com/theoria-project/theoria.git
cd theoria
pip install -e ".[dev]"
```

## Verify Installation

```bash
theoria version
```
