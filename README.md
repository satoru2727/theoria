# Theoria

Humanities research & LaTeX drafting agentic CLI.

This is free and unencumbered software released into the public domain. Do What The Fuck You Want To.

## Tech Stack

- Python 3.11+
- LangGraph (agent orchestration)
- LiteLLM (provider abstraction)
- Typer + Rich (CLI)

## Install

```bash
pip install -e ".[dev]"
```

## Usage

```bash
theoria --help
theoria init
theoria chat
```
